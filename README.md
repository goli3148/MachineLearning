## How Should I Use It?
first of all create a virtual venv: <br/>
`> py -m venv .venv`<br />
after that enable the venv:<br/>
`> .venv\scripts\activate.ps1`<br/>
or<br/>
`> .venv\scripts\activate.bat`<br/>
then install the requerements:<br/>
`> py -m pip install -r requirements.txt`<br/>
